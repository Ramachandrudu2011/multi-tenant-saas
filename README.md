# Multi-Tenant SaaS Project

## Developed By
**Ramachandrudu Chakali**

## Project Overview
This is a Multi-Tenant SaaS (Software as a Service) web application developed as a full-stack project. It allows multiple tenants to use the same application while keeping their data separate and secure.

## Features
- User Authentication
- Multi-Tenant Architecture
- Dashboard
- Login System
- Protected Routes
- Responsive User Interface
- React Frontend
- Node.js Backend
- REST API Integration

## Technologies Used
### Frontend
- React.js
- Vite
- JavaScript
- HTML
- CSS

### Backend
- Node.js
- Express.js

## Installation

### Clone Repository
```bash
git clone https://github.com/Ramachandrudu2011/multi-tenant-saas.git
```

### Install Dependencies
```bash
npm install
```

### Run Frontend
```bash
cd client
npm run dev
```

### Run Backend
```bash
cd server
npm start
```
